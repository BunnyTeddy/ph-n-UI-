## Database Structure (Firestore)

```
users/
  {userId}/
    - email
    - name
    - createdAt
    
plants/
  {plantId}/
    - userId
    - name
    - species
    - plantedDate
    - imageUrl
    - createdAt
    
diary_entries/
  {entryId}/
    - plantId
    - userId
    - content
    - images[]
    - createdAt
    
sensor_data/
  {plantId}/
    - temperature
    - soilMoisture
    - timestamp
```
